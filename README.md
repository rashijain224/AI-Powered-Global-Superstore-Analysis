# Global Superstore Sales Analysis

## About the Project

This project analyzes the Global Superstore dataset to understand sales performance, profitability, customer behavior, and regional trends. The analysis was completed using SQL, Microsoft Excel, Power BI, ChatGPT, and Microsoft Copilot.

The goal was not only to answer business questions but also to present the findings in a way that would help stakeholders make informed decisions.

---

## Project Objectives

- Perform Exploratory Data Analysis (EDA)
- Identify sales and profit trends
- Analyze customer segments and regional performance
- Answer business questions using SQL
- Build Pivot Tables and charts in Excel
- Design a Power BI dashboard
- Provide business insights and recommendations

---

## Key Findings

- Technology generated the highest sales and profit among all product categories.
- Consumer was the largest customer segment.
- APAC contributed the highest overall sales.
- Standard Class was the most frequently used shipping mode.
- Some sub-categories generated high sales but relatively low profit due to higher discounts.
